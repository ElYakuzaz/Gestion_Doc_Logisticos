# Gestion_Doc_Logisticos
Proyecto de estadia para el proyecto de gestion de documentos logisticos utilizando la API de Filebound

---- 
* DJANGO commands to get proyect running
py --version
py -m django --version

-- create proyect:
py -m django startproject myproject
-- run the server
py manage.py runserver

-- inside folder start app PARA CREAR UN PROGRAMA NUEVO (front, back, api, etc)
py manage.py startapp app

---- 
-- api de filebound
